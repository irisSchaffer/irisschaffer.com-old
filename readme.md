# irisschaffer.com

## Requirements

- [Node](https://nodejs.org/en/download) > 5.0
- [Yarn](https://yarnpkg.com/en/docs/install)

## Install

```bash
$ yarn
```

```bash
$ cp .env.example .env
```

## Development

```bash
$ yarn develop # will start the development server on port 3000
```

Note that setting a different port is currently not supported.

## Production

```bash
$ yarn build:render
$ yarn serve
```
